Smartydev
Smartdev is a repository for list of tools to be built on HNG Internship 7.0 track.

Installation
Use the git pull request pull to install Smartydev.

echo "# smartydev" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/gbengaayelab/smartydev.git
git push -u origin master
Usage
Fork the repository and make your commits to a separate branch
